## Project Setup

```sh
Clone and Run the Laravel API first then start the below process [Laravel configuration](https://github.com/joeljeremiahrupiah/coalition-tech-laravel-api)
Install node js to your computer first
Clone the project with -> git clone https://github.com/joeljeremiahrupiah/coalition-tech-vue-frontend.git
To install dependencies run -> npm install
To serve the project run -> npm run dev
This will be your local url to test the frontend functionality [Local Vue/Vite Url](http://localhost:5174/)
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
